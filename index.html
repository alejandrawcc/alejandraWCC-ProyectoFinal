Método de Newton-Raphson

Búsqueda de raíces

Problema práctico:

Cálculo del punto de equilibrio en un modelo económico donde:

Utilidad(x) = Ingresos(x) - Costos(x)

Donde: Ingresos(x) = 50x - 0.5x² y Costos(x) = 100 + 10x + 0.1x³

Buscamos la producción x donde la utilidad es cero (punto de equilibrio).

Método de Jacobi

Sistemas lineales

Problema práctico:

Resolver el sistema de ecuaciones que modela el flujo de corriente en un circuito eléctrico:

10I₁ - 2I₂ - 4I₃ = 12

-2I₁ + 8I₂ - 2I₃ = 0

-4I₁ - 2I₂ + 10I₃ = 0

Donde I₁, I₂, I₃ son las corrientes en amperios.

Serie de Taylor

Aproximación de funciones

Problema práctico:

Aproximar la función cos(x) usando series de Taylor para calcular el coseno de un ángulo en radianes.

La serie de Taylor para cos(x) alrededor de 0 es:

cos(x) ≈ 1 - x²/2! + x⁴/4! - x⁶/6! + ...

Útil en procesamiento de señales y gráficos por computadora donde se necesitan cálculos trigonométricos eficientes.

Resultados:

Presiona "Calcular Aproximación" para ver los resultados
